조건부 렌더링 아래 사이트의 코드를 이용하여 클래스 컴포넌트를 만들어 본다. 참고: https://reactjs.org/docs/conditional-rendering.html Greeting 컴포넌트에서 isLoggedIn 의 값에 따라 리턴하는 컴포넌트 분기한다. isLoggedIn 은 부모로 부터 받는 값이다. 부모로부터 받은 props 를 자식 컴포넌트에 넘기시오. Greeting 컴포넌트에서 UserGreeting 컴포넌트로 모든 props를 넘기시오. Greeting 컴포넌트에서 GuestGreeting 컴포넌트로 모든 props를 넘기시오. Greeting 컴포넌트의 인자로 isLoggedIn 을 넘긴다. isLoggedIn 값으로는 true 를 넘기도록 설정하시오.
